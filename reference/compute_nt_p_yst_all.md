# Compute all summary scores for nt_p_yst.

This function computes all summary scores for the nt_p_yst form. Make
sure to have all necessary columns in the data frame.

## Usage

``` r
compute_nt_p_yst_all(data)
```

## Arguments

- data:

  tbl. Data frame containing the columns to be summarized.

## Value

tbl. The input data frame with the summary scores appended as new
columns.

## Examples

``` r
if (FALSE) { # \dontrun{
compute_nt_p_yst_all(data)
} # }
```
