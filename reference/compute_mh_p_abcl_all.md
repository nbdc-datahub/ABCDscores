# Compute all summary scores for mh_p_abcl.

This function computes all summary scores for the mh_p_abcl form. Make
sure to have all necessary columns in the data frame.

## Usage

``` r
compute_mh_p_abcl_all(data)
```

## Arguments

- data:

  tbl. Data frame containing the columns to be summarized.

## Value

tbl. The input data frame with the summary scores appended as new
columns.

## Examples

``` r
if (FALSE) { # \dontrun{
compute_mh_p_abcl_all(data)
} # }
```
