# Compute all summary scores for su_y_nicsre.

This function computes all summary scores for the su_y_nicsre form. Make
sure to have all necessary columns in the data frame.

## Usage

``` r
compute_su_y_nicsre_all(data)
```

## Arguments

- data:

  tbl. Data frame containing the columns to be summarized.

## Value

tbl. The input data frame with the summary scores appended as new
columns.

## Examples

``` r
if (FALSE) { # \dontrun{
compute_su_y_nicsre_all(data)
} # }
```
