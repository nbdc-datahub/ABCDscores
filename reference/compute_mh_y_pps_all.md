# Compute all PPS scores

This super function computes all scores in PPS using all the **default**
arguments.

## Usage

``` r
compute_mh_y_pps_all(data)
```

## Arguments

- data:

  tbl, Dataframe containing the columns to be summarized.

## Value

tbl. The input data frame with the summary scores appended as new
columns.

## Details

Make sure the `data` is the full set of all variables from PPS

## Examples

``` r
if (FALSE) { # \dontrun{
compute_mh_y_pps_all(data)
} # }
```
